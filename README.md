# wfkj-work

## 《Vue 大事件》项目（学习版）

通过学习《Vue 大事件》项目，巩固 Vue 基础中所学的知识，掌握从 0 到 1 开发企业级项目的能力。

## 所需的前置知识

1. Vue 基础

- vue-cli、指令、组件、vue-router路由、vuex

2. axios

- baseURL、拦截器

3. element-ui

- 安装与配置、常用的组件

4. npm 与 模块化

## 安装两个软件

> vscode 和 node

```
node -v
npm -v
```
